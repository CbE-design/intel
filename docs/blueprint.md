# **App Name**: Veritas Intel

## Core Features:

- Subject Profile Management: Create, update, and manage subject profiles with detailed personal information and identifiers.
- Background Check Initiation: Initiate background checks with configurable parameters and criteria specific to South African regulations.
- Report Generation: Generate comprehensive background check reports with analyzed data and risk assessments. The system uses a tool to help determine whether particular information should be incorporated into the report, depending on the background check parameters and South African regulations.
- Interactive Location Mapping: Display consented mobile device locations on an interactive map with historical tracking data.
- Search and Filtering: Enable advanced search and filtering capabilities across all subject profiles and reports.

## Style Guidelines:

- Primary color: Deep blue (#2E4A62) to convey security and professionalism.
- Background color: Dark gray (#333333) for a clear and secure interface.
- Accent color: Light gray (#A9A9A9) for text, borders, and highlights, ensuring readability against the dark background.
- Headline font: 'Inter', sans-serif, for a modern and machined look; suitable for headlines and body text.
- Use crisp, minimalist icons for navigation and data representation.
- Maintain a clean, structured layout with clear visual hierarchy to ensure ease of use.
- Implement subtle animations for loading states and interactive elements.
